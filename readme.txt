# IG Voice RAG Chatbot

## Project Overview
A Retrieval-Augmented Generation (RAG) chatbot that provides contextual responses based on video scripts.

## Features
- Video script-based question answering
- Strict adherence to source material
- Quirky goat-themed responses

## Setup

### Prerequisites
- Python 3.8+
- pip

### Installation
1. Clone the repository
```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/IG-Voice-RAG-Test.git
cd IG-Voice-RAG-Test
```

2. Create a virtual environment
```bash
python3 -m venv venv
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

## Running the Application
```bash
python app.py
```

## Configuration
Rename `.env.example` to `.env` and fill in your configuration details.
